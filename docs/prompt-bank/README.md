# Prompt Bank

High-performance, battle-tested prompts for AI agents and LLMs.

## What is Prompt Bank?

Prompt Bank is a curated collection of effective prompts organized by use case. Each prompt is optimized for clarity, specificity, and reliable output.

## Categories

Prompts are categorized by:
- Task type (generation, analysis, refactoring, etc.)
- Domain (code, documentation, data processing)
- Complexity level

## Usage

Each prompt includes:
- Clear objective statement
- Input format requirements
- Expected output structure
- Example usage

Browse the collection to find prompts that fit your workflow, or use them as templates for custom variations.
