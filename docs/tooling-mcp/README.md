# Tooling & MCP

MCP server configurations and custom tool definitions for AI agents.

## What is MCP?

Model Context Protocol (MCP) provides a standardized way to connect AI agents to external tools, data sources, and services.

## Contents

This section includes:
- MCP server configurations
- Custom tool definitions
- Integration patterns
- Connection examples

## Getting Started

MCP enables AI agents to interact with:
- File systems
- Databases
- APIs and web services
- Development tools
- Custom business logic

Each tool definition includes schema, usage examples, and configuration guidance.
