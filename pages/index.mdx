# Entropy Wiki

**Cyber-utilitarian monorepo for AI skills, prompts, and MCP toolsets**

Capture the chaos of real systems, compress it, and ship it as repeatable advantage.

## Architecture

- **[Beads](/beads)** - Atomic logic snippets and issue tracking system
- **[Gastown](/gastown)** - Multi-agent orchestration and rig standards
- **[Skills Bank](/skills-bank)** - Functional capabilities
- **[Prompt Bank](/prompt-bank)** - Categorized, high-performance prompts
- **[Tooling MCP](/tooling-mcp)** - MCP server configs and custom tool definitions
- **[Orchestration](/orchestration)** - Multi-agent handoff protocols
- **[Context](/context)** - MAP.md strategies and token optimization
- **[Lab](/lab)** - Learnings, failure logs, model-specific quirks

## Quick Start

```bash
# Clone the repository
git clone https://github.com/wboudy/entropy-wiki.git
cd entropy-wiki

# Install dependencies
npm install

# Run development server
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to view the wiki.

## Contributing

This project uses **beads** for issue tracking:

```bash
bd ready              # Find available work
bd show <id>          # View issue details
bd update <id> --status=in_progress  # Claim work
bd close <id>         # Complete work
```
